# Personal Website (Jekyll)

Concise researcher-style personal site + blog, built with Jekyll and GitHub Pages.

## Quick Customize

- Profile: edit `_data/profile.yml`
- Navigation: edit `_data/navigation.yml`
- Publications: edit `_data/publications.yml`
- Styling: edit `assets/css/main.css`
- Posts: add Markdown files to `_posts/`

## Local Preview

Prereqs: Ruby 3+ and Bundler.

1. Install dependencies:
   - `bundle install`
2. Run the dev server:
   - `bundle exec jekyll serve`
3. Open:
   - `http://localhost:4000`
